# PES 2021 Classic Teams Mod

Proyecto para crear equipos históricos en PES 2021 mediante un pipeline automatizado.

## Estructura
- data/raw: datos sin procesar (no versionados)
- data/processed: datos limpios y versionados
- scripts: scraping, procesamiento y build
- config: reglas y configuración

## Filosofía
- Raw data no se versiona
- Processed data sí
- Pipeline reproducible
